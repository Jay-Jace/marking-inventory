## 2. 대시보드

로그인 후 첫 화면입니다.

### 창고별 재고 현황
상단에 3개 창고의 현재 재고가 카드 형태로 표시됩니다.

| 카드 | 설명 |
|------|------|
| 🏪 오프라인샵 | 오프라인 매장 현재 재고 |
| 🏭 플레이위즈 | 제작창고(플레이위즈) 현재 재고 |
| 🚚 CJ창고 | CJ대한통운 물류창고 현재 재고 |

- 각 카드를 보면 SKU별 수량을 한눈에 파악할 수 있습니다.

### 진행 중인 작업지시서
- 현재 진행 중인 오더 목록이 표시됩니다.
- **상태별 단계**: 업로드됨 → 이관준비 → 이관중 → 입고확인완료 → 마킹중 → 마킹완료 → 출고완료

## 3. 작업지시서 업로드

왼쪽 메뉴 **"작업지시서 업로드"** 클릭

### 업로드 전 준비사항
- BERRIZ에서 다운로드한 **출고수량 엑셀 파일** (.xlsx) 준비
- BOM이 등록되어 있어야 마킹단품 자동 분류 가능 (→ [BOM 관리](#5-bom-관리) 먼저 설정)

### 업로드 절차

**Step 1.** 파일 선택
- 화면 중앙의 업로드 영역을 클릭하거나 파일을 드래그&드롭
- BERRIZ에서 받은 `출고수량` 시트가 포함된 엑셀 파일 선택

**Step 2.** 파싱 결과 확인
- 시스템이 자동으로 파일을 분석합니다.
- **마킹 필요 품목**: BOM에 등록된 완제품 → 마킹 작업 필요
- **마킹 불필요 품목**: BOM 없는 단품 → 마킹 없이 바로 출고

**Step 3.** 저장
- 내용 확인 후 **"작업지시서 저장"** 버튼 클릭
- 저장 완료 시 대시보드에 작업지시서가 표시됨

### ⚠️ 주의사항
- 동일 날짜의 작업지시서를 중복 업로드하지 않도록 주의하세요.
- BOM이 등록되지 않은 완제품은 마킹 불필요 품목으로 처리됩니다.

## 4. 양식 다운로드 (BERRIZ 업로드용)

왼쪽 메뉴 **"양식 다운로드"** 클릭

BERRIZ에 업로드해야 하는 6가지 엑셀 양식을 단계별로 다운로드합니다.
**모든 BERRIZ 업로드는 관리자가 직접 수행합니다.**

---

### STEP 1 — 작업지시서 업로드 직후 즉시 다운로드

> 오프라인 매장에서 플레이위즈로 재고를 이관하기 전에 BERRIZ에 등록

#### ① 이관 오더 (Transfer Order)
- **용도**: 오프라인샵 → 플레이위즈 재고 이동 등록
- **내용**: 이관할 완제품 SKU · 수량
- **다운로드 후**: BERRIZ 이관 오더 메뉴에 업로드

#### ② 재고조정 — 오프라인샵 M차감
- **용도**: 오프라인샵 재고에서 마킹용 수량 차감 처리
- **내용**: 오프라인샵 SKU · 차감 수량 (M타입)
- **다운로드 후**: BERRIZ 재고조정 메뉴에 업로드

> 💡 STEP 1 양식은 작업지시서 업로드 직후 바로 활성화됩니다.

---

### STEP 2 — 플레이위즈 입고 확인 완료 후 다운로드

> 플레이위즈 담당자가 "입고 확인" 완료 후 활성화됨

#### ③ 재고조정 — 플레이위즈 P증가
- **용도**: 플레이위즈(제작창고) 재고 증가 처리
- **내용**: 플레이위즈 단품 SKU · 입고 수량 (P타입)
- **다운로드 후**: BERRIZ 재고조정 메뉴에 업로드

> 💡 플레이위즈에서 "입고 확인" 버튼을 눌러야 이 양식이 활성화됩니다.

---

### STEP 3 — 마킹 작업 완료 후 다운로드

> 플레이위즈 담당자가 전체 마킹 완료 처리 후 활성화됨

#### ④ 재고조정 — 플레이위즈 M차감
- **용도**: 플레이위즈 재고에서 마킹 완료 수량 차감
- **내용**: 플레이위즈 단품 SKU · 차감 수량 (M타입)

#### ⑤ CJ창고 입고의뢰 (G타입)
- **용도**: CJ물류창고로 마킹 완료 제품 입고 의뢰
- **내용**: CJ창고 SKU · 수량 (G타입)

#### ⑥ 제작 입고의뢰 (P타입)
- **용도**: 제작 완료 제품 BERRIZ 입고 등록
- **내용**: 완제품 SKU · 수량 (P타입)

> 💡 STEP 3의 3개 양식은 동시에 다운로드하여 BERRIZ에 업로드합니다.

---

### 다운로드 버튼 상태 안내

| 상태 | 표시 | 의미 |
|------|------|------|
| 초록색 활성 | **다운로드** 버튼 | 클릭하여 엑셀 다운로드 가능 |
| 회색 비활성 | **잠금** 표시 | 이전 단계 미완료, 아직 다운로드 불가 |

## 5. BOM 관리

왼쪽 메뉴 **"BOM 관리"** 클릭

BOM(Bill of Materials): 완제품 1개를 구성하는 단품 목록입니다.
예) 유니폼 세트(완제품) = 상의 단품 + 하의 단품

### BOM 엑셀 파일 형식

| 열 | 내용 | 예시 |
|----|------|------|
| A | 완제품 SKU ID | `UNI-2024-001` |
| B | 완제품 SKU명 | `삼성라이온즈 홈 유니폼 세트 L` |
| C | 단품 SKU ID | `TOP-2024-001` |
| D | 단품 SKU명 | `삼성라이온즈 홈 상의 L` |
| E | 수량 | `1` |

### BOM 업로드 절차

**Step 1.** BOM 엑셀 파일 준비 (위 형식 준수)

**Step 2.** 화면에서 파일 선택 또는 드래그&드롭

**Step 3.** 파싱 결과 확인 후 **"BOM 저장"** 클릭

**Step 4.** 하단 목록에서 완제품별 BOM 구성 확인

### BOM 삭제
- 목록에서 삭제할 항목 우측 **삭제** 버튼 클릭
- 해당 완제품-단품 연결이 제거됩니다.

### ⚠️ 주의사항
- BOM은 작업지시서 업로드 **전에** 등록되어 있어야 합니다.
- BOM이 없는 완제품은 마킹 불필요 품목으로 자동 처리됩니다.

## 6. 전체 워크플로우 요약

```
[관리자] BOM 등록
    ↓
[관리자] BERRIZ 출고수량 엑셀 → 작업지시서 업로드
    ↓
[관리자] STEP 1 양식 다운로드 → BERRIZ 업로드
    (① 이관 오더 + ② 재고조정 오프라인 M차감)
    ↓
[오프라인매장] 재고 발송 확인 (발송 완료 버튼)
    ↓
[플레이위즈] 입고 수량 확인
    ↓
[관리자] STEP 2 양식 다운로드 → BERRIZ 업로드
    (③ 재고조정 플레이위즈 P증가)
    ↓
[플레이위즈] 마킹 작업 완료 입력 (매일 진척 입력)
    ↓
[관리자] STEP 3 양식 다운로드 → BERRIZ 업로드
    (④ 플레이위즈 M차감 + ⑤ CJ 입고의뢰 + ⑥ 제작 입고의뢰)
    ↓
완료
```

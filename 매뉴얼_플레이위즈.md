## 1. 로그인

1. 브라우저에서 서비스 URL 접속
2. **이메일**: 담당자 이메일 주소 입력
3. **비밀번호**: 설정된 비밀번호 입력
4. **로그인** 버튼 클릭 → 입고 확인 화면으로 자동 이동

> 💡 플레이위즈 계정은 **입고 확인**과 **마킹 작업** 메뉴만 표시됩니다.

---

## 2. 입고 확인

왼쪽 메뉴 **"입고 확인"** 클릭 (로그인 시 자동 이동)

### 언제 사용하나요?
오프라인 매장에서 재고를 발송 완료한 후, 플레이위즈에 실제로 도착한 수량을 확인할 때 사용합니다.

### 화면 구성

| 항목 | 설명 |
|------|------|
| SKU명 | 입고된 단품 이름 |
| SKU ID | 단품 고유 코드 |
| 예정 수량 | 오프라인 매장에서 보내기로 한 수량 |
| 실제 입고 수량 | 실제로 받은 수량 (직접 입력) |

### 입고 확인 절차

**Step 1.** 도착한 실물 재고 수량 확인
- 오프라인 매장에서 보낸 단품들을 수령하고 수량을 셉니다.

**Step 2.** 실제 입고 수량 입력
- 각 SKU의 실제 입고 수량을 입력란에 입력합니다.
- 예정 수량과 다를 경우 실제 받은 수량을 정확히 입력하세요.

**Step 3.** 입고 확인 완료 버튼 클릭
- 모든 수량 입력 후 **"입고 확인 완료"** 버튼 클릭
- 이 시점부터 관리자가 STEP 2 양식을 다운로드할 수 있습니다.

### ⚠️ 주의
- 예정 수량과 실제 수량이 다를 경우 반드시 실제 수량으로 입력하세요.
- 관리자에게 수량 불일치 내용을 알려주세요.
- 한 번 확인 완료 처리 후에는 수정이 불가합니다.

---

## 3. 마킹 작업

왼쪽 메뉴 **"마킹 작업"** 클릭

### 언제 사용하나요?
입고 확인 완료 후, 매일 마킹 작업이 끝날 때마다 당일 완료 수량을 입력합니다.

### 화면 구성

| 항목 | 설명 |
|------|------|
| SKU명 | 마킹 작업 중인 단품 이름 |
| 총 입고 수량 | 입고 확인 완료된 전체 수량 |
| 기완료 수량 | 이전 날까지 마킹 완료된 누적 수량 |
| 잔여 수량 | 아직 마킹하지 않은 수량 |
| 오늘 완료 수량 | 오늘 마킹 완료한 수량 (직접 입력) |

### 마킹 작업 입력 절차

**Step 1.** 당일 마킹 작업 완료
- 실제 마킹 작업을 진행합니다.

**Step 2.** 오늘 완료 수량 입력
- 각 SKU별로 오늘 마킹 완료한 수량을 입력합니다.
- 잔여 수량을 초과하여 입력할 수 없습니다.

**Step 3.** 저장 버튼 클릭
- **"마킹 완료 저장"** 버튼 클릭
- 잔여 수량이 0이 되면 해당 오더의 상태가 **마킹완료**로 변경됩니다.

---

## 4. 미완료 수량 자동 이월

당일에 전체 수량을 마킹하지 못해도 됩니다.

### 이월 방식
- 오늘 입력한 수량만큼 누적 완료 수량이 쌓입니다.
- **내일 마킹 작업 화면에 자동으로 잔여 수량이 표시됩니다.**
- 별도 조작 없이 잔여 수량이 계속 관리됩니다.

### 예시

| 날짜 | 총 수량 | 기완료 | 오늘 입력 | 잔여 |
|------|---------|--------|-----------|------|
| 1일차 | 100 | 0 | 40 | 60 |
| 2일차 | 100 | 40 | 35 | 25 |
| 3일차 | 100 | 75 | 25 | 0 ✅ |

3일차에 잔여 수량이 0이 되면 해당 SKU의 마킹이 완료됩니다.

## 5. 주의사항

- **입고 확인** 전에는 마킹 작업 화면에 항목이 표시되지 않습니다.
  → 입고 확인을 먼저 완료하세요.

- 마킹 완료 수량은 **당일 실제 완료한 수량만** 입력하세요.
  계획 수량이 아닌 실제 수량 기준입니다.

- 전체 오더의 모든 SKU가 마킹완료 상태가 되어야 관리자가 STEP 3 양식을 다운로드할 수 있습니다.

- 입력 오류 발생 시 관리자(jace.nice@kakaoent.com)에게 연락하세요.

## 1. 로그인

1. 브라우저에서 서비스 URL 접속
2. **이메일**: 담당자 이메일 주소 입력
3. **비밀번호**: 설정된 비밀번호 입력
4. **로그인** 버튼 클릭 → 발송 확인 화면으로 자동 이동

> 💡 오프라인 매장 계정은 **발송 확인** 메뉴만 표시됩니다.

---

## 2. 발송 확인 화면

로그인 후 자동으로 발송 확인 화면이 열립니다.

### 화면 구성

#### 오늘 플레이위즈로 보내야 할 재고 목록

관리자가 작업지시서를 업로드하면, 플레이위즈로 이동해야 할 단품 목록이 표시됩니다.

| 항목 | 설명 |
|------|------|
| SKU명 | 발송할 단품 이름 |
| SKU ID | 단품 고유 코드 |
| 발송 수량 | 플레이위즈로 보내야 할 수량 |
| 현재 재고 | 현재 오프라인 매장 보유 수량 |
| 상태 | 재고 충분 / 재고 부족 |

#### 재고 부족 경고
- 발송 수량보다 현재 재고가 부족한 경우 **빨간색 경고**가 표시됩니다.
- 재고 부족 시 관리자에게 즉시 연락하세요.

---

## 3. 발송 완료 처리

### 절차

**Step 1.** 발송 목록 확인
- 화면에 표시된 품목과 수량을 실물 재고와 대조합니다.

**Step 2.** 실물 재고 포장 및 발송 준비
- 목록에 표시된 수량대로 단품을 준비하여 플레이위즈로 발송 준비합니다.

**Step 3.** 발송 완료 버튼 클릭
- 실제 발송이 완료된 후 화면 하단의 **"발송 완료 확인"** 버튼을 클릭합니다.
- 클릭 즉시 플레이위즈에서 입고 확인을 할 수 있는 상태로 변경됩니다.

### ⚠️ 주의
- **발송 완료 버튼은 실제 물건을 보낸 후에만 누르세요.**
- 한 번 클릭하면 되돌릴 수 없습니다. 관리자에게 연락이 필요합니다.

---

## 4. 주의사항

- 작업지시서가 없으면 발송 목록이 표시되지 않습니다.
  → 관리자에게 작업지시서 업로드를 요청하세요.
- 재고 부족 품목이 있으면 관리자와 협의 후 처리하세요.
- 발송 전날 미리 재고 수량을 확인해두면 당일 작업이 빠릅니다.
